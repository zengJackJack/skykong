# 广州星控科技有限公司 — 企业 Landing Page 设计文档

## 项目概述

为广州星控科技有限公司（Guangzhou Xingkong Technology Co., Ltd.）设计并开发一个高端、大气的企业 Landing Page，以 GitHub Pages 纯静态页面形式托管。页面需同时面向中国及海外市场客户，展示公司旗下的两款核心产品：

- **CutControl** — 电子开料锯上位机控制系统
- **Optimizer** — 板材优化下料软件

## 技术方案

- **部署方式**: GitHub Pages（静态托管）
- **技术栈**: 纯静态 HTML + CSS + JavaScript（单页应用）
- **语言支持**: 自动检测浏览器语言 + 手动切换（中文 / English），数据存储在全局 i18n JSON 对象中
- **响应式**: 全面适配桌面 / 平板 / 手机
- **第三方依赖**: 无（纯原生实现，以保证 GitHub Pages 兼容性和加载速度）

## 设计语言

### 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | `#0B0F1A` | 深邃海军蓝黑 |
| 卡片/区块背景 | `#131B2E` | 深蓝半透明，玻璃态效果 |
| 主色（Primary） | `#00B4FF` | 亮青蓝，科技感光效，用于标题/链接/装饰 |
| 辅色（Accent） | `#FF6B35` | 暖橙，CTA 按钮/高亮/数据高亮 |
| 主文字 | `#FFFFFF` | 白色标题/主要文字 |
| 辅文字 | `#8892B0` | 柔灰蓝，正文/副标题 |
| 分割线/边框 | `#1E2A4A` | 深蓝边框 |

### 字体系统

- **标题**: `'Inter', 'Noto Sans SC', sans-serif`（无衬线，现代感）
- **正文**: `'Inter', 'Noto Sans SC', sans-serif`
- 英文字体使用 Inter，中文字体使用 Noto Sans SC，保证中英文混排时的视觉协调

### 交互特性

- 纯 CSS 平滑滚动 (`scroll-behavior: smooth`)
- 滚动渐入动画（Intersection Observer API，元素进入视口时触发 fadeIn + translateY）
- Canvas 粒子/网格背景动画（Hero 区域）
- 导航栏吸附效果（`position: sticky` + 背景透明→实心过渡）
- CTA 按钮悬停发光效果
- 卡片悬停上浮效果

## 区块详细设计

### 1. 导航栏 (Navbar)
- 固定在页面顶部，背景随滚动从透明过渡到半透明深色
- 左侧：公司 Logo（`homelogo.jpg`）
- 右侧：产品（下拉菜单显示 CutControl / Optimizer）、关于、联系、🌐 语言切换按钮
- 移动端折叠为汉堡菜单

### 2. Hero 区
- 全屏高度（`100vh`），Canvas 绘制的动态粒子网络背景
- 粒子颜色：`#00B4FF`，连线半透明白，缓慢浮动
- 中央内容：
  - 大标题：广州星控科技有限公司（巨大，带 text-shadow 发光效果）
  - 英文名：Guangzhou Xingkong Technology Co., Ltd.
  - Slogan：智能板材加工解决方案 / Intelligent Panel Processing Solutions
  - 装饰分割线（主色发光）
  - 两个 CTA 按钮：了解更多（主色填充）、查看产品（透明边框）
- 底部：↓ 滚动指示动画（弹跳箭头）

### 3. 关于我们
- 左右两栏布局（桌面），移动端上下排列
- 左栏：公司介绍（先中后英），介绍公司专注于板材加工智能化
- 右栏：几何图形占位（CSS 绘制的六边形/线条网格，主色调，后续可替换为实拍图）
- 下方：四个数据指标卡片
  - 10+ 年行业经验 / 10+ Years Experience
  - 500+ 服务客户 / 500+ Clients
  - 99% 客户满意率 / 99% Satisfaction
  - 5+ 国家地区 / 5+ Countries

### 4. CutControl 产品区

- 右侧：产品名 + 简短描述 + 核心特性列表（6项）
  - 多品牌 PLC 支持 (Modbus TCP / Beckhoff ADS)
  - 5 种行业格式导入 (SAW/PTX/HD/XML/MT)
  - 实时 3D 仿真可视化

  - 中英多语言界面
- 下方：6 个功能特性卡片（3×2 网格），每卡有图标 + 标题 + 简短说明

### 5. Optimizer 产品区
